Q&R / Guide

Comment ajouter un membre ?
Cliquez sur le bouton "+" en haut à droite. Sélectionnez le type de lien (Branche, Enfant ou Conjoint) et remplissez les détails.
Comment modifier un membre ?
Touchez la bulle d'un membre pour le centrer. Si c'est un membre que vous avez ajouté, touchez sa bulle (ou son nom) pour le modifier.
Comment naviguer dans l'arbre ?
Touchez une bulle pour entrer dans cette lignée. Utilisez le bouton "Retour" pour remonter.
Signaler une erreur ?
Utilisez le bouton contact ci-dessus en précisant le nom du membre concerné.
